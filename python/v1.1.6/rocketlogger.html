get_time(absolute_time=False, time_reference='local')[source]¶

Get the timestamp of the data.

Using simple linear interpolation to generating the sample from the block timestamps.

Parameters

  • absolute_time – Wether the returned timestamps are relative to the start time in seconds (default) or absolute timestamps

  • time_reference –

    Which clock data to get (for absolute time only, i.e. when absolute_time=True)

    • ’local’ The local oscillator clock (default)

    • ’network’ The network synchronized clock

Returns

A numpy array containing the timestamps

load_file(filename, join_files=True, decimation_factor=1, memory_mapped=True)[source]¶

Read a RocketLogger data file and return an RLD object.

Parameters

  • filename – The filename of the file to import. If numbered files following the “<filename>_p#.rld” convention are found, they can be joined during import using the join_files flag.

  • join_files – Enalbe joining of multiple files if numbered files following the “<filename>_p#.rld” convention are found.

  • decimation_factor – Decimation factor for values read

  • memory_mapped – Set False to fall back to read entire file to memory at once, instead of using memory mapped reading. Might increase file read performance for many smaller files and/or some system configurations.
